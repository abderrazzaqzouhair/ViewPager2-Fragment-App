# ViewPager2-Fragment-App

## Description

This project, part of the Android Component, demonstrates the use of ViewPager2 to create an application that features three distinct fragments.

- **Fragment 1** provides an introduction to the app, detailing its functionalities and purpose.
- **Fragment 2** showcases a list of elements (such as cities or countries) using RecyclerView, featuring at least five items.
- **Fragment 3** presents an image accompanied by descriptive text.

The project highlights navigation between fragments through horizontal swiping, showcasing fundamental concepts in Android development and user interface design.

## Features

- MainActivity with ViewPager2
- Three fragments:
  - Introduction fragment with app description
  - RecyclerView displaying a list of elements
  - Fragment with an image and descriptive text
- Smooth navigation between fragments
